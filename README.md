# 🚀 SpaceFlow CFD Solver

A Python-based Computational Fluid Dynamics (CFD) solver developed from scratch for learning and simulating fundamental fluid flow equations.

## 📌 Features

- Uniform 2D Mesh Generation
- Gaussian Pulse Initial Condition
- 2D Linear Advection Solver
- 2D Diffusion Solver
- Convection-Diffusion Solver
- Multiple Boundary Conditions
- Real-Time Animation
- Modular Project Structure

## 🚀 How to Run

Clone the repository

```bash
git clone <repository-link>
```

Go into the project

```bash
cd SpaceFlow-CFD
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```
